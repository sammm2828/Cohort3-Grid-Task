## 📝 About The Assignment

This project is part of **A4 - Assignment Four**. The objective was to choose any three design components from a given complex, modern UI (Bento Box style) and perfectly recreate them using **CSS Grid Layout**. 

The chosen designs feature a dark mode aesthetic with vibrant lime green and purple accents, showcasing complex layouts like profile cards, navigation menus, and statistics.

## 🛠️ Technical Requirements Met

To successfully complete this assignment, the following CSS techniques and properties were strictly implemented:
* ✔️ `display: grid` for the core layout structure.
* ✔️ `grid-template-columns` & `grid-template-rows` for precise sizing, spanning, and positioning of the cards.
* ✔️ `gap` for uniform and responsive spacing between the grid items.
* ✔️ `position: relative` & `position: absolute` for internal component alignment (such as badges, decorative icons, and overlapping elements).
* ✔️ Proper spacing, padding, and alignment techniques to ensure a pixel-perfect match with the provided design.
